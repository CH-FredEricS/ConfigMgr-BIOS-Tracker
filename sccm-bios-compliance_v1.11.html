Run this script on any machine with direct SQL access to the ConfigMgr site database. It exports device BIOS data as a JSON file.

# ConfigMgr BIOS Export Script # Requires: Read-Only Analyst role (or SQL read access) param( [string]$SiteServer = "CMSERVER", [string]$SiteCode = "P01", [string]$OutputFile = ".\cm_bios_export.json" ) $conn = New-Object System.Data.SqlClient.SqlConnection $conn.ConnectionString = "Server=$SiteServer;Database=CM_$SiteCode;Integrated Security=SSPI;Connect Timeout=30" $conn.Open() # Check whether Win32_ComputerSystemProduct is enabled in Hardware Inventory. # When enabled, its Version0 field contains the SMBIOS Type 1 version string. # For HP devices this includes "SBKPFV3" if the BIOS is already prepared for # the new Secure Boot certificates (per HP ISH_13070353). $chkCmd = $conn.CreateCommand() $chkCmd.CommandText = "SELECT COUNT(*) FROM sys.views WHERE name = 'v_GS_COMPUTER_SYSTEM_PRODUCT'" $hasCspView = [int]$chkCmd.ExecuteScalar() -gt 0 if ($hasCspView) { Write-Host "Win32_ComputerSystemProduct view found — CSPVersion will be included." -ForegroundColor Cyan } else { Write-Host "Win32_ComputerSystemProduct not found in HW Inventory — CSPVersion will be empty." -ForegroundColor Yellow Write-Host "To enable: CM Console → Administration → Client Settings → Hardware Inventory → Add class → Win32_ComputerSystemProduct" -ForegroundColor Yellow } # Build query dynamically: include CSP join only when the view exists $cspSelect = if ($hasCspView) { ",`n csp.Version0 AS CSPVersion" } else { ",`n NULL AS CSPVersion" } $cspJoin = if ($hasCspView) { "`nLEFT JOIN v_GS_COMPUTER_SYSTEM_PRODUCT csp ON sys.ResourceID = csp.ResourceID" } else { "" } $query = @" SELECT DISTINCT sys.Name0 AS ComputerName, cs.Manufacturer0 AS Manufacturer, cs.Model0 AS Model, bios.SMBIOSBIOSVersion0 AS BIOSVersion, bios.ReleaseDate0 AS BIOSReleaseDate, hw.LastHWScan AS LastHardwareScan$cspSelect FROM v_R_System sys JOIN v_GS_COMPUTER_SYSTEM cs ON sys.ResourceID = cs.ResourceID JOIN v_GS_PC_BIOS bios ON sys.ResourceID = bios.ResourceID LEFT JOIN v_GS_WORKSTATION_STATUS hw ON sys.ResourceID = hw.ResourceID$cspJoin WHERE sys.Client0 = 1 AND sys.Operating_System_Name_and0 LIKE '%Windows%' ORDER BY sys.Name0 "@ $cmd = $conn.CreateCommand(); $cmd.CommandText = $query; $cmd.CommandTimeout = 120 $reader = $cmd.ExecuteReader(); $results = @() while ($reader.Read()) { $results += [PSCustomObject]@{ ComputerName = $reader["ComputerName"] Manufacturer = $reader["Manufacturer"] Model = $reader["Model"] BIOSVersion = $reader["BIOSVersion"] BIOSReleaseDate = if ($reader["BIOSReleaseDate"] -is [DBNull]) { $null } else { $reader["BIOSReleaseDate"].ToString("yyyy-MM-dd") } LastHardwareScan = if ($reader["LastHardwareScan"] -is [DBNull]) { $null } else { $reader["LastHardwareScan"].ToString("yyyy-MM-dd HH:mm") } CSPVersion = if ($reader["CSPVersion"] -is [DBNull]) { $null } else { $reader["CSPVersion"] } } } $reader.Close(); $conn.Close() $results | ConvertTo-Json -Depth 3 | Out-File $OutputFile -Encoding UTF8 Write-Host "Exported $($results.Count) devices to $OutputFile" -ForegroundColor Green
Required views v_R_System · v_GS_COMPUTER_SYSTEM · v_GS_PC_BIOS · v_GS_WORKSTATION_STATUS — the built-in Read-Only Analyst role is sufficient.

Optional: v_GS_COMPUTER_SYSTEM_PRODUCT — enables the SBKPFV3 Secure Boot readiness check for HP devices. Enable via Administration → Client Settings → Hardware Inventory → Set Classes → Win32_ComputerSystemProduct.
⚠️ BIOSVersion is only populated if Hardware Inventory is enabled and devices have reported. Devices with no BIOS data will show as Not checked.

Create a custom SSRS report using the query below, then export as CSV (comma delimited) from the report viewer.

-- SSRS Report Dataset Query -- Optional: remove the CSPVersion line and its JOIN below if -- Win32_ComputerSystemProduct is not enabled in Hardware Inventory. SELECT DISTINCT sys.Name0 AS ComputerName, cs.Manufacturer0 AS Manufacturer, cs.Model0 AS Model, bios.SMBIOSBIOSVersion0 AS BIOSVersion, CONVERT(varchar(10), bios.ReleaseDate0, 120) AS BIOSReleaseDate, CONVERT(varchar(16), hw.LastHWScan, 120) AS LastHardwareScan, ISNULL(csp.Version0, '') AS CSPVersion -- SMBIOS Type 1 version; HP: contains SBKPFV3 if BIOS is Secure Boot ready FROM v_R_System sys JOIN v_GS_COMPUTER_SYSTEM cs ON sys.ResourceID = cs.ResourceID JOIN v_GS_PC_BIOS bios ON sys.ResourceID = bios.ResourceID LEFT JOIN v_GS_WORKSTATION_STATUS hw ON sys.ResourceID = hw.ResourceID LEFT JOIN v_GS_COMPUTER_SYSTEM_PRODUCT csp ON sys.ResourceID = csp.ResourceID -- remove if not enabled WHERE sys.Client0 = 1 AND sys.Operating_System_Name_and0 LIKE '%Windows%' ORDER BY sys.Name0
Export tip In the SSRS report viewer: Actions → Export → CSV (comma delimited). Then load that file in Step 2.
